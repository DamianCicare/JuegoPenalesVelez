# ⚽ Vélez Penales Mobile ⚪🔵

¡Bienvenido a **Vélez Penales Mobile**! Un juego de simulación de penales desarrollado en Python utilizando la librería **Pygame**. Este proyecto fue creado con un cariño especial para mi hijo Fabriccio, fanático del Fortín.

### 🎮 Sobre el Juego

El juego pone a prueba tus reflejos pateando penales en una tanda de penales emocionante. Está diseñado pensando en la portabilidad, con una configuración estable para dispositivos móviles(futuro) y detección automática de pantalla completa.

**Características principales:**
* **Ambiente de Estadio:** Incluye sonidos de hinchada, silbato inicial y efectos de gol para una experiencia inmersiva.
* **Optimización Mobile:** Ajuste automático de resolución y configuración de audio estable (`pygame.mixer`).
* **Lógica Aleatoria:** El portero cuenta con un sistema de decisión inteligente basado en aleatoriedad para mayor dificultad.

---

### 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 🐍
* **Librería Principal:** [Pygame](https://www.pygame.org/)
* **Módulos Adicionales:** `os`, `random`
* **Assets:** Imágenes y sonidos personalizados (WAV/JPG).

### 🚀 Instalación y Ejecución

Si quieres probar el juego en tu PC, sigue estos pasos:
``` 
1. **Clona el repositorio:**
bash

2. **Instala las dependencias:**
Asegúrate de tener Python instalado y luego ejecuta:

Bash
pip install pygame
Corre el juego:

Bash
python juego.py
```

📁 Estructura de Archivos
juego.py: El núcleo del juego (lógica, renderizado y eventos).

juego.spec: Archivo de configuración para empaquetar el juego (PyInstaller).

/assets: Archivos de sonido (gol.wav, hinchada.wav, silbato.wav) y el escudo oficial.
